01 Node Modules Error

In this sample we are going to setup the basic plumbing to "build" our project and launch it in a dev server.

But we got an error because we aren't excluding node_modules folder.

To fix this build, we have to add "node_modules" row in exclude section. 

                
                  "exclude": [
                     "node_modules",
                     "typings/main",
                     "typings/main.d.ts"
                  ],

We are going to use:

  • Webpack

  • Typescript

Highlights

The most interesting parts worth to take a look
